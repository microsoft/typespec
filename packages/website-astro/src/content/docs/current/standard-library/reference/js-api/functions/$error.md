```ts
$error(context, entity): void
```

`@error` decorator marks a model as an error type.

`@error` can only be specified on a model.

## Parameters

| Parameter | Type |
| :------ | :------ |
| `context` | [`DecoratorContext`](../interfaces/DecoratorContext.md) |
| `entity` | [`Model`](../interfaces/Model.md) |
