```ts
function $name(
   context, 
   target, 
   name): void
```

Provide the name of the XML element or attribute. This means the same thing as
 `@encodedName("application/xml", value)`

## Parameters

| Parameter | Type |
| ------ | ------ |
| `context` | `DecoratorContext` |
| `target` | `Type` |
| `name` | `string` |

## Returns

`void`
