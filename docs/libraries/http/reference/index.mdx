## Install

<Tabs>
<TabItem value="spec" label="In a spec" default>

```bash
npm install @typespec/http
```

</TabItem>
<TabItem value="library" label="In a library" default>

```bash
npm install --save-peer @typespec/http
```

</TabItem>
</Tabs>
